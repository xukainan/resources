设计版式定义

 

“设计版式”是指网站的外观视觉效果，包括页面的配色方案、整体布局、内容排版、主

图设计、

页面尺寸大小等几方面元素，

每种设计版式与上述元素属于唯一对应关系，

即指

每种设计版式对应唯一的配色方案、整体布局、内容排版、主图设计及页面尺寸大小，如

需对其中某种元素进行变更，

无论变更程度大小，

则当理解为不同的设计版式。

页面内容

信息不属于设计版式范畴，内容信息变更不应包含在设计版式的定义中。

 



 

“相似性版式”是指设计版式中的其他元素保持不变的情况下，对内容排版、主图

设计作出变更；

 



 

“相异性版式”是指设计版式中的主要元素都可能存在差异。

 

通常情况下，

若甲方需要就同一级别的页面进行不同的设计，

则本附件中的上一部分则必

须注明需要设计的相似或者相异性版式的数量，

若未注明，

则本附件中上一部分中所指的

版式是指“相异性版式”

，若数量为

1

，则说明针对该页面进行唯一的版式设计。
